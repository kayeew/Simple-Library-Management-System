Simple Library Management System
--------------------------------
Kayee

Setup:
------
1. Run XAMPP
2. Start Apache and MySQL
3. Copy library project folder to your XAMPP htdocs folder

Setup database: 
4. Open localhost/phpmyadmin
5. Create a new database named "library"
6. import library.sql to the database

Or change database connection settings in:
\library\includes\dbcon.php

7. Open localhost/library browser
